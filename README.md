# Springboot-Vue-Blog

&emsp;这是前后端分离项目，尚且有很多需要完善.

### 线上展示

&emsp;[线上效果展示](https://106.12.211.47)

### 后端技术栈

1. springboot
2. mybatis-plus
3. swagger
4. druid
5. jwt

### 前端技术栈

1. Vue3
2. vue-router
3. vuex
4. element-plus


### 项目运行

1. 配置完环境后
2. 先运行后端项目，运行blog-api的SpringbootBlogApplication
3. 在运行前端项目，进入my-blog-app执行 yarn run serve
